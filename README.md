# 🌿 Plant Disease Detection using Deep Learning

A deep learning-based image classification project that identifies healthy and diseased plant leaves using CNNs and Transfer Learning. The model is trained on the augmented PlantVillage dataset containing 87K+ images across 38 plant disease classes.

## 🚀 Features

- Image classification of 38 plant disease classes
- Exploratory Data Analysis (EDA)
- Image preprocessing & augmentation
- Custom CNN and Transfer Learning models
- Model evaluation using standard metrics
- Ready for deployment with FastAPI

## 📂 Dataset

- **Dataset:** PlantVillage (Augmented)
- **Images:** ~87,000 RGB Images
- **Classes:** 38
- **Split:** 80% Train | 20% Validation | Test Images
